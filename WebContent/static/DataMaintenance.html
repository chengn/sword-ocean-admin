添加SQL脚本

添加参数有二种写法如下：

第一种写法：参数直接写在脚本中，那么参数使用?代替，同时【添加参数】，输入参数名称和参数对应脚本中？的序号，从1开始。

写在SQL脚本中的方式： select * from test where name=? and com = ?
添加参数：{name = 1},{com = 2}

第二种写法：参数不写在SQL脚本中，使用添加参数的方式写入。 则对应的参数带有对应的sql，这种方式可以通过参数组装sql

SQL脚本输入如： select * from test where 1
请点击添加参数如： [{'name'='1','content'='2'}]
下面可添加多个参数
